#  TechStore Sales Analysis Using SQL

## Project Overview

This project demonstrates how SQL can be used to analyze sales data for a fictional electronics retail company called **TechStore**.

The project was completed using MySQL Workbench and includes database design, table creation, data insertion, SQL analysis, and business recommendations.

This project forms part of my Data Analytics Portfolio.

## Business Questions Answered

This project answers the following business questions using SQL:

- Who are the highest-spending customers?
- Which products generate the most revenue?
- Which city generates the highest revenue?
- What is the average order value for each customer?
- Which customers spent more than R10,000?
- Which products are the most expensive?
